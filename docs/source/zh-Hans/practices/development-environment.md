## 基础环境

### 安装 [Node.js](https://nodejs.org)

运行命令来测试 Node.js 是否安装：

```bash
$ node -v
```

运行命令来测试 NPM 是否安装：
```bash
$ npm -v
```

### 安装 [Yarn](https://yarnpkg.com/zh-Hans/)

运行下列命令来安装 Yarn：

```bash
$ curl -o- -L https://yarnpkg.com/install.sh | bash
```

运行命令来测试 Yarn 是否安装：

```
$ yarn --version
```

### 安装 [Java](http://www.oracle.com/technetwork/java/javase/overview/index.html)

```
$ java -version
```
